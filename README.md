# GSE301 Python Project – Student Academic Performance System

## Project Overview
This project is a console-based Python system designed to store, process, and analyze mock academic data for a university department. It demonstrates core Python concepts such as data structures, conditional logic, error handling, pattern matching, and interactive menus.

## Features Implemented
- Student data storage using lists, dictionaries, tuples, and sets
- Grade calculation using IF–ELIF–ELSE and MATCH CASE
- Input validation with TRY–EXCEPT
- List slicing and set operations
- Graduation eligibility checker using logical operators
- Interactive menu system using MATCH CASE

## Technologies Used
- Python 3.10+
- VS Code / Jupyter Notebook
